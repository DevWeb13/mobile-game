# mobile-game

A playable HTML prototype for the Token Clash timing game.

## Run locally

```bash
cd /workspace/mobile-game
python3 -m http.server 8000
```

Open <http://localhost:8000/index.html> in your browser.

## One-command option

```bash
./start.sh
```

## Controls
- Select a stake tier.
- Click **Start Round** to begin.
- Press **Tap!** (or Spacebar) to stop the indicator and score.
